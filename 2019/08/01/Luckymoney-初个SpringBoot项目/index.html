    我是用的是使用 IDEA 的 自带的 Spring Initializr，也可以使用线上版本的 Spring Initializr。

    主要是添加 Jpa和MySQL 的依赖。

    以及提供一些对数据的基本操作。

    5. 创建LuckmoneyController

    Controller 主要是处理跟用户交互的，一般每个业务逻辑都会有一个 Controller，提供给用户 http 请求接口，用户请求接口进行数据访问。

    既然要和用户交互，那么免不了要使用到数据库。

    这里使用Jpa进行数据库的访问。

    LuckymoneyRepository.java

    要使用jpa只需要继承一个JpaRepository类，并指明实体类和ID的类型即可。
